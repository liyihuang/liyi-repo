## 一、我的赌注（买入前必须写清楚）

```
1. 我为什么买：
   猪价9.7元处于十年新低，全行业巨亏，产能正在出清。
   牧原成本11.5元，是行业最低，猪价回到13元牧原就盈利，二线还在亏。

2. 市场在担心什么：
   - 散户出清导致周期消失：认为中国会像美国一样规模化后没有猪周期。
   - 需求被白肉替代：中国人对猪肉消耗在下降，大家都吃白肉和蛋白了。
   - 能繁去化被PSY对冲：PSY（每头母猪年提供仔猪）从18涨到28，能繁减了但小猪可能没少。
   - 38元已经price in了反弹预期。

3. 我认为市场错了，因为：
   - 【关于周期消失】：美国规模化后周期没有消失，只是变长了（3-4年拉长至7-11年）且波幅收窄（10-25%）。美国是在1988-2000年间完成小户清退的（年出栏<1000头的占比从32%降至2%）。而中国目前散户（500头以下）仍占35%左右，类似于美国1990年的整合起点。这意味着中国目前仍处于“最后一轮高波动大周期”的窗口内。
   - 【关于需求替代】：美国过去50年健康观念升级，鸡肉人均消费暴涨4倍，但人均猪肉消费稳定在~50磅（横盘了50年，并非总量崩溃）。猪周期是供给侧驱动的，只要供需能出清，存量博弈依然能带来价格暴涨。
   - 【关于PSY提升】：虽然PSY提升会部分对冲能繁下降（意味着名义去化幅度需要打折看），但去化依然在发生，且我们可以通过同步监控PSY数据来修正预测。
   - 【关于估值】：38元只隐含了猪价回到14.5元（以 normalized PE 10x 估算）的极低预期。如果猪价反弹到18-20元，空间依然巨大。

4. 我错了的信号：
   - 能繁不降反升，或PSY飙升导致实际生猪供给预测不减反增（去化逻辑失败）
   - 30个月猪价仍未站上14元（时间逻辑失败）
   - 牧原自身出重大问题（公司逻辑失败）
```

## 二、每月跟踪清单（每月1号，15分钟）

### 必看（3个数据）

| 数据 | 来源 | 怎么看 |
|------|------|--------|
| **生猪价格** | 搜"今日猪价"或"外三元猪价" | 记录当月均价，和上月对比 |
| **能繁母猪存栏** | 农业农村部月度数据，搜"能繁母猪存栏 最新" | 记录数字，看环比涨还是跌 |
| **PSY（生产效率）** | 东方财富/萝卜投研等查最新券商周报中的“养殖效率”，或涌益咨询/上海钢联数据 | 看行业平均PSY是否继续飙升。若PSY暴涨，能繁去化幅度需要打折扣 |

### 观察（不用专门查，看到就记）

| 数据 | 怎么看 |
|------|--------|
| **二线企业新闻** | 有没有新的破产/重整/退市消息 → 有 = 去化加速，利好 |
| **牧原股价对利空的反应** | 坏消息出来当天，收红还是收绿？（利空不跌是底部见底信号） |

## 五、卖出规则（不测顶、趋势跟踪离场）

目标是收益最大化，不是卖在最高点。策略只有三件事：持有、持有、进入高估区后趋势破位走人。

### 核心逻辑

```
高估区之前 → 什么都不做，不管涨了多少
进入高估区 → 启动趋势跟踪
趋势破位   → 无条件全清
```

中途股价涨涨跌跌、小周期震荡、利润回吐——全部无视。只有两件事能让你卖出：趋势破位，或者逻辑坏了（见第六节止损规则）。

---

### 第一步：高估区门槛（何时启动跟踪）

满足任意一条即进入高估区：

```
□ PB > 4.0x（对应股价约 60 元）
□ 外三元猪价 > 18 元/kg 持续 1 个月
```

**在此之前，无论股价涨到哪里，不启动跟踪，不卖出。**

---

### 第二步：趋势跟踪参数

进入高估区后，只看一个东西：

```
每周五记录收盘价
追踪这波行情的历史最高周收盘价

触发条件：某周五收盘价 < 最高周收盘价 × 0.80（回撤 20%）
操作：下周一开盘无条件全仓清空
没有例外，没有”再看看”
```

用周线而非日线：过滤日常噪音，牧原在大牛市中途回调最大约 18%，真正见顶后第一波跌幅超过 25%。20% 周线是平衡点——能拿住中途震荡，又能抓住真正的转向。

---

### 关于小周期的处理

如果猪价只涨到 14-15 元、股价到 50 出头，始终没进高估区：

- 不启动跟踪，不卖出
- 看能繁：还在降 → 什么都不做，等下一波；止降或回升 → 大周期可能就这么大了
- 最差结果是坐了一段过山车，但成本在底部，时间是你的朋友

**小周期调整中唯一需要看的指标是能繁母猪存栏：**

```
还在降 → 供给仍在收缩，价格波动是噪音，不动
持平   → 观察，每月确认方向
回升   → 周期可能已到头，结合猪价位置重新判断
```

PSY 每季度校准一次（能繁降但 PSY 飙升 = 实际产能去化打折）。

---

### 持有 18 个月解锁点

如果 18 个月后仍未进入高估区（猪价和 PB 都没到门槛），允许自己做一次重新评估：当前全局配置，这笔钱放在这里是不是最优的？

这不是止损点，是一个解锁点——18 个月之前，不去想机会成本，专心持有。

## 六、止损规则（放弃条件）

### 逻辑止损（链条断了）

```
信号 1：能繁存栏连续3个月环比回升，且猪价仍低于12元
信号 2：行业PSY因疫病改良或繁育爆发而大幅跃升，导致出栏生猪供给不降反升
含义：供给层面的去化出清逻辑宣告失败
操作：减仓一半，观察3个月，如果继续回升则清仓
```


### 时间止损（等太久了）

```
信号：持有满30个月，猪价仍未站上14元
含义：周期比预期长得多
操作：重新评估——还愿意再等2年吗？不确定就减半
```

### 公司止损（牧原自身出事）

```
信号：牧原疫病大爆发/财务造假/重大治理问题
操作：卖出牧原，如果仍然看好猪周期则换成猪ETF
```

## 九、预期管理

```
最好情况：猪价反弹到18-20元，股价到70-90元，2-3年赚80-130%
基本情况：猪价反弹到14-16元，股价到50-60元，2年赚30-60%
最差情况：猪周期钝化，猪价在10-13元磨，30个月后减仓离场，亏10-20%
黑天鹅：牧原疫病/造假，亏30-50%后止损
```

**我能接受最差情况吗？** → 如果能，执行计划。如果不能，缩小仓位。
